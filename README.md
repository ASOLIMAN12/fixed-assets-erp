# Fixed Assets ERP Pro

نظام إدارة الأصول الثابتة باللغة العربية، يدعم:

- تسجيل دخول وصلاحيات مستخدمين.
- استعادة كلمة المرور عبر البريد الإلكتروني.
- إدارة الأصول: إضافة / تعديل / حذف / نقل مركز تكلفة.
- أصل مشترك بين أكثر من مركز تكلفة مع توزيع الإهلاك بالنسب.
- استيراد وتصدير Excel حسب ملف الأصول المعتمد.
- قيد إهلاك شهري بنفس شكل الشيت المعتمد.
- تجميع قيد الإهلاك حسب مركز التكلفة + المجموعة.
- سجل عمليات Audit Log.

## بيانات الدخول الافتراضية

- اسم المستخدم: `admin`
- كلمة المرور: `123456`

## التشغيل المحلي

```bash
npm install
npm start
```

ثم افتح:

```text
http://localhost:3000
```

## إعداد استعادة كلمة المرور

1. انسخ ملف `.env.example` إلى ملف جديد باسم `.env`.
2. ضع بيانات SMTP الخاصة بك.
3. في Gmail استخدم App Password وليس كلمة مرور البريد الأساسية.
4. لا ترفع ملف `.env` إلى GitHub.

البريد الافتراضي لاستعادة كلمة المرور:

```text
ahmededitor245@gmail.com
```

يمكن تغييره من `.env` في المتغير:

```text
ADMIN_RECOVERY_EMAIL=
```

## ملفات Excel المعتمدة

ضع القوالب الرسمية داخل مجلد:

```text
templates/
```

القوالب الحالية:

- ملف الأصول الثابتة المعتمد.
- ملف قيد الإهلاك الشهري المعتمد.

## قبل رفع المشروع على GitHub

- اجعل المستودع Private.
- لا ترفع ملف `.env`.
- لا ترفع أي ملف يحتوي على كلمات مرور أو بيانات SMTP.
- اختبر الاستيراد والتصدير قبل الاعتماد النهائي.
